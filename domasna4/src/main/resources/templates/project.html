Navigation Application

This is a web application for navigating to Fast Food and Pub offering places in the central area of Skopje.

For the architecture of this application was used Spring Boot as backend framework and Thymeleaf for getting data from backend. For the map we used Leaflet map and it's functions. The frontend was implemented using Bootstrap classes and for the functionality on the page we used Javascript and jQuery.
